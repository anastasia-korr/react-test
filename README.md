# Setup manual

1. Clone repository
2. Open project directory 
```sh
$ cd ./path-to-project/react-test
```
3. Install dependencies 
```sh
$ npm install
```
```sh
$ bower install
```
4. Run webpack to build application
```sh
$ webpack
```
5. Run local server
```sh
$ npm run start
```
6. Open http://localhost:3000 in your browser
